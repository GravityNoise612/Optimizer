🚀 QUICK START (3 STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Create Repository
   → Go to https://github.com/new
   → Name it: gaming-settings-optimizer
   → Make it PUBLIC
   → Click "Create repository"

2. Push Files
   → git clone https://github.com/YOURUSERNAME/gaming-settings-optimizer.git
   → Copy all files from here to the cloned folder
   → git add .
   → git commit -m "Initial commit"
   → git push origin main

3. Enable GitHub Pages
   → Go to Settings → Pages
   → Source: Deploy from branch
   → Branch: main / (root)
   → Click Save
   → Wait 1-2 minutes
   → Visit: https://YOURUSERNAME.github.io/gaming-settings-optimizer/

🎮 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. User opens index.html
2. App detects their hardware automatically
3. User selects their game folder
4. Scanner finds installed games by their .exe files
5. User clicks on a game to see recommended settings
6. Data is saved to browser's local storage
